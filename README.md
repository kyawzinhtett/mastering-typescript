### Initialize TypeScript Compiler

`tsc --init`

### Compile all TypeScript Files

`tsc`

### Useful Settings

-   `"target": "ES2016"`,
-   `"rootDir": "./src"`,
-   `"outDir": "./dist"`,
-   `"removeComments": true`,
-   `"noEmitOnError": true`,
-   `"sourceMap": true`,
-   `"noImplicitAny": true (Use with caution)`,
-   `"noUnusedParameters": true`,
-   `"noImplicitReturns": true`,
-   `"noUnusedLocals": true`,
-   `"allowUnreachableCode": false`,
-   `"noImplicitOverride": true`
